util

Business-domain-independent framework code. This is code that could be used for any Java project - not just DS3. No dependencies on any business logic or specific product lines are allowed here. May not have any dependencies on other components.

common

Business-domain-dependent framework code. This is code that is specific to the DS3 product line and includes code intended to be re-used across the product line's other components. May have a dependency on the util component.

Includes database dao definitions and RPC contracts between the DS3 datapath frontend and other components, such as the tape and pool backends, as well as other DS3 datapath frontend processes.

The RPC contracts defined herein are the canonical authority. Please ensure they are kept up-to-date.
